# SokobanDemo3
## 介绍

本次项目为考研且AC66小组团队项目，借助easyx这一c++图形库，并且做了相对复杂的图形动画效果（小人走路，每次走路包含4帧等）。

当时写项目还比较青涩，虽然是小组项目。版本控制使用的qq群文件（笑）

***
项目内容：图形化推箱子。

环境：vs2019。
